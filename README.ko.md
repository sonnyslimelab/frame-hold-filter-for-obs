# 🎬 OBS용 Frame Rate Reducer

OBS Studio에서 영상의 출력 프레임 빈도를 조절하여 저프레임(Low FPS) 효과를 만들어주는 필터 플러그인입니다.

## ✨ 주요 기능

- 🎞️ 60FPS 영상을 30FPS, 15FPS, 10FPS 같은 저프레임 스타일로 변환
- 🔧 원하는 프레임 간격으로 효과 조절 가능
- 🎥 OBS Studio 필터로 간편하게 적용
- ⚡ 실시간 방송 환경에 적합한 가벼운 구조

## 🚀 설치 방법

1. 최신 Release 파일 다운로드
2. OBS Studio 설치 폴더에 파일 압축 해제
3. OBS Studio 재시작
4. 소스의 필터 목록에서 "Frame Rate Reducer" 추가

## 💡 활용 예시

- 레트로 게임 / 오래된 카메라 느낌 연출
- 일부러 끊기는 듯한 영상 효과
- CCTV 효과
